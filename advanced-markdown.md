# Advanced Markdown

---

### **1. Tables**
Create tables using `|` and `-`:

| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Row 1    |
| Row 2    | Row 2    |

### **2. Task Lists**

Use `- []` for task lists.

- [ ] Task 1

- [x] Task 2

### **3. Code Blocks**
Wrap code in triple backticks (```) for syntax highlighting:

```python
def hello_world():
    print("Hello, world!")
```

### **4. Blockquotes**
Use `>` for blockquotes:

> This is a quote.
